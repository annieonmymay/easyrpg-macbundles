== INSTALLATION ==
Pour lancer le jeu, double cliquez sur l'icone "RPG_RT.EXE" dans le dossier principal
Une fois le jeu lancé appuyez sur F4 pour jouer en mode fenêtre
Utilisez F5 pour faire varier la taille de la fenêtre

Si un message d'erreur apparait, il va certainement falloir télécharger et installer un programme du nom de RPG Maker 2003, il n'est pas très lourd et permet de créer des RPG facilement! Il est assez facile à trouver en français sur internet, mais il faut aussi télécharger un autre programme qui s'appelle les RTP, sans lequel RPG Maker ne peut fonctionner. Faites bien attention d'installer la version 2003. Ensuite réessayez l'icone RPG_RT.EXE

Si ça ne marche toujours pas, je ne sais pas quoi faire pour vous. Allez demander de l'aide à des gens plus intelligents que moi (il y en a beaucoup) ou essayez d'utiliser un autre ordinateur... ou faites un tour dans la rubrique "problèmes et solutions", quelque part plus loin dans ce fichier. Sans doute.

== CONTROLES ==
Utilisez les flèches directionelles pour déplacer le personnage principal (le batteur)
Utilisez entrée ou espace pour interagir avec des éléments du décor ou d'autres personnages
Utilisez Esc ou 0 (zéro) pour ouvrir le menu principal et pour revenir en arrière dans les menus.
Utilisez la touche F4 pour passer du mode plein écran au mode fenêtre, et F5 en mode fenêtre pour changer la taille de cette dernière.

PROBLEMES ET SOLUTIONS
====================================================================================================

== LE JEU NE DEMARRE PAS ==
OFF a été conçu sous RPGMaker2003 et ne tourne normalement que sous Windows. Si vous n’utilisez pas Windows, il faudra lancer le jeu à partir d’un émulateur (Wine par exemple, existe pour Linux et Mac OS).
Dans de rares cas, il est nécessaire d’installer RPGMaker 2003 ainsi que les RTP.

== LE JEU NE SAUVEGARDE PAS LA PARTIE ==
Si vous avez téléchargé une version de OFF compressée dans un fichier ZIP ou RAR, décompressez l’intégralité du fichier dans un dossier unique (clic droit sur l’archive puis "Décompresser tout") puis supprimez l’archive ZIP ou RAR et lancez le jeu normalement en double cliquant sur RPG_RT.EXE.
Sous Windows 7, il est possible que le jeu n’ait pas les droits nécessaires pour enregistrer des fichiers sur le disque dur de votre ordinateur. Tout d’abord, assurez-vous le jeu n’a pas été décompressé dans "Program Files" ou "Program Files x86". Si ça n’a pas résolu le problème, essayez de lancer l’application avec les droits administrateur (clic droit sur le fichier RPG_RT.EXE puis "lancer en tant qu’administrateur").

== L'ECRAN SCINTILLE AU DEMARRAGE ==
Par défaut, OFF se lance ne mode plein écran en 320x240. Si votre carte graphique ne permet pas l’affichage plein écran dans cette résolution, passez en mode fenêtré (en appuyant sur F4 ou en appuyant simultanément sur Alt+Enter). Si l’image est trop petite, vous pouvez doubler la taille des pixels en appuyant sur F5.

== LES CONTROLES CLAVIER NE REPONDENT PAS ==
Il arrive parfois que certains périphériques branchés sur la machine entrent en conflit avec le jeu, débranchez les contrôleurs de jeu (type pad, joystick, etc.) et relancez le jeu.

== LES SONS FONCTIONNENT, MAIS PAS LES MUSIQUES ==
Téléchargez LAME MP3 Encoder et décompressez les fichiers MP3 contenus dans le dossier "Music" vers du WAV (attention : décompresser les fichiers MP3 nécessite plus d’espace disque).

== LE TEXTE EST ILLISIBLE ==
Copiez le fichier "ProFont.fon" (il se trouve dans le dossier d’installation de OFF) dans le dossier font de Windows ("C:\Windows\fonts"  par défaut). Sur certaines versions de Windows, il est possible d’installer la police de caractère en faisant un clic droit dessus puis "installer la police".
